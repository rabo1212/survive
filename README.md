# 🎓 종강의 복수: 서바이버

뱀파이어 서바이버 스타일의 생존 게임!

## 🎮 조작법
- **WASD / 방향키** : 이동
- **공격은 자동!**

## 🎯 게임 목표
- 5분 동안 생존하면 승리!
- 교수님들을 처치하고 경험치를 모아 레벨업!
- 레벨업할 때마다 새로운 무기와 스탯을 선택!

## 🔫 무기 종류
- 📚 전공책 - 기본 무기
- ✏️ 연필 - 빠른 연사
- ☕ 커피 - 관통
- 💻 노트북 - 고데미지
- 🧽 지우개 - 다중 발사

## 👾 적 종류
- 🧑‍🏫 조교 - 약함
